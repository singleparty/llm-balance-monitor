# Status Bar

A VS Code extension that displays info in the status bar.

## Features

- Adds a clickable status bar item that shows the current time.
- Command: `Status Bar: Show Time`.

## Development

```bash
npm install
npm run compile
```

Press `F5` in VS Code to launch the Extension Development Host.

## Scripts

- `npm run compile` — build TypeScript to `out/`
- `npm run watch` — build in watch mode
- `npm run lint` — run ESLint
